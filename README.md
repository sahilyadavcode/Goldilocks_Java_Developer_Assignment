Goldilocks Java Assignment
📘 Description
This repository contains solutions to the Java programming assignment provided by Goldilocks. The tasks are designed to evaluate practical Java development skills across multiple domains, simulating real-world scenarios that a developer may encounter in enterprise projects.

🎯 Objectives
Demonstrate proficiency in core Java concepts such as string manipulation, collections, and command-line input.

Apply file handling techniques to read/write CSV files, compute sums, and detect duplicates.

Connect to a MySQL database using JDBC and perform queries.

Implement networking tasks including HTTP requests and building a simple HTTP server.

Design a basic frontend login interface with validation.

Write SQL queries to filter and retrieve data.

Practice unit testing and QA by writing test case scenarios for each program.

🛠️ Deliverables
Java source code for each of the 10 tasks.

SQL scripts for database setup and queries.

Unit test scenarios (JUnit or manual test cases).

Documentation explaining how to run each program.
